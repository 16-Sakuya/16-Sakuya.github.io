实验五：远程管理

实验目标

  • 安装和配置OpenSSH服务，实现多机之间的远程登录管理
  • 配置SSH密钥认证，实现免密登录
  • 了解多机开发环境下的账户管理方法
  • 掌握使用SCP和rsync进行跨机文件传输与同步

实验环境

  • 上一实验的两台虚拟机(node1, node2)，且处于同一Host-Only网络

检查SSH服务状态

配置SSH服务器

  • 编辑/etc/ssh/sshd_config
    sudo nano /etc/ssh/sshd_config
  • 重启服务：sudo systemctl restart ssh

生成SSH密钥对（node1）

ssh-keygen -t ed25519 -C "邮箱"

  • 生成后查看

将公钥复制到远程服务器

ssh-copy-id misaka@192.168.56.103

  • misaka@192.168.56.103是node2

测试免密登录：ssh misaka@192.168.56.103

使用SCP传输文件

  • exit回到node1


  • scp (-r) 上传的文件/目录 远端服务器:下载到的目录
  • scp (-r) 远端服务器:下载的文件/目录

使用rsync同步文件

  • 安装rsync：sudo apt install rsync -y
